# Поисковый движок «AйТиБокс»

Консольное приложение для поиска по документам, реализующее инвертированный индекс и ранжирование по релевантности.

## Стек технологий
- C++17
- CMake
- nlohmann/json — для работы с JSON
- Google Test — для модульного тестирования

## Структура
- `config.json` — настройки и список файлов
- `requests.json` — поисковые запросы
- `answers.json` — результаты поиска
- `resources/` — текстовые документы

## Запуск тестов
Тесты подключены через Google Test. Запускаются автоматически при сборке
